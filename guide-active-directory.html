FAQ

Quelle version de Windows Server est couverte ?

Le guide est compatible Windows Server 2019 et 2022. Les captures d'écran sont basées sur Windows Server 2022.

Faut-il du matériel spécifique ?

Non. Le guide inclut une section pour monter un lab virtuel gratuit (Hyper-V ou VirtualBox) pour s'entraîner sans risque.

Le guide parle-t-il d'Azure AD / Entra ID ?

Un chapitre de synthèse présente la complémentarité AD on-premises / Entra ID pour les PME hybrides.
