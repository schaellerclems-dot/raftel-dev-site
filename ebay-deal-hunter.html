Est-ce légal de scraper eBay ?

Le script utilise les données publiques d'eBay de manière raisonnable. Il respecte les délais entre requêtes pour ne pas surcharger les serveurs.

Non. Le script consulte les annonces publiques sans nécessiter de connexion.

Le calcul est basé sur le prix d'achat et un prix de revente estimé que vous définissez. C'est un indicateur — vérifiez toujours avant d'acheter.
