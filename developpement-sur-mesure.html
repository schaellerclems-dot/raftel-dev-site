Questions fréquentes

Vous avez des questions ?

Comment se déroule le paiement ?

50% à la commande, 50% à la livraison. Pour les projets plus courts, le paiement intégral à la livraison est possible. Paiement par virement bancaire ou PayPal.

Travaillez-vous avec des clients hors de Normandie ?

Oui, 100% en télétravail. Les échanges se font par e-mail, visio ou messagerie selon votre préférence. Je travaille avec des clients partout en France et à l'international.

Quels sont les délais de réalisation ?

Un site vitrine simple : 5 à 10 jours. Une application web : 2 à 6 semaines. Un script Python : 2 à 7 jours. Les délais sont précisés dans le devis et respectés.

Est-ce que je reçois le code source ?

Oui. Vous recevez systématiquement le code source complet, hébergé sur un dépôt GitHub privé transféré à votre compte à la livraison. Le projet vous appartient entièrement.

Proposez-vous de la maintenance après livraison ?

Un support de 30 jours est inclus après livraison pour corriger d'éventuels bugs. Pour une maintenance longue durée ou des évolutions, un devis complémentaire est établi.

Mon projet est-il trop petit ou trop grand ?

Aucun projet n'est trop petit — une simple page ou un script de 50 lignes sont les bienvenus. Pour les projets très complexes (équipe entière, architecture distribuée), je peux vous orienter vers les ressources adaptées.
